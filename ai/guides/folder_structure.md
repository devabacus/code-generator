# Структура папок AI-агентов

Полная структура каталога для работы с ИИ-агентами.

---

## 📁 Дерево каталогов

```
📁 ai/
├── 📄 README.md            — Главный README
├── 📄 version.md           — Версионирование
│
├── 📁 docs/                — Документация
│   ├── 📁 decisions/       — Архитектурные решения (ADR)
│   │   └── 📄 adr-0001-template.md
│   ├── 📄 INDEX.md         — Индекс документации
│   ├── 📄 agent_memory.md  — Gotchas, предпочтения, заметки
│   ├── 📄 architecture.md  — Архитектура проекта
│   ├── 📄 conventions.md   — Соглашения и стандарты
│   ├── 📄 dev_guide.md     — Руководство разработчика
│   ├── 📄 roadmap.md       — Дорожная карта (фазы)
│   ├── 📄 status.md        — Текущий статус
│   ├── 📄 troubleshooting.md — Решение проблем
│   └── 📄 workflow.md      — Описание рабочего процесса
│
├── 📁 guides/              — Дополнительные руководства
│   ├── 📄 folder_structure.md  — Этот файл
│   ├── 📄 user_guide.md        — Руководство пользователя
│   └── 📄 worktree_guide.md    — Git Worktree
│
├── 📁 prompts/             — Промпты для агентов
│   ├── 📄 finalize.prompt.md   — Завершение дискуссии
│   ├── 📄 teamlead.prompt.md   — Промпт тимлида
│   └── 📄 executor.prompt.md   — Промпт исполнителя
│
├── 📁 scripts/             — Скрипты автоматизации
│   └── 📄 new_task.py      — Создание новой задачи
│
└── 📁 tasks/               — Управление задачами
    ├── 📁 _template/       — Шаблоны задач
    │   ├── 📄 report.md    — Шаблон отчёта
    │   └── 📄 task.md      — Шаблон задачи
    ├── 📁 active/          — Активные задачи
    ├── 📁 blocked/         — Заблокированные задачи
    ├── 📁 done/            — Выполненные задачи
    └── 📄 backlog.md       — Бэклог задач
```

---

## 📋 Назначение папок

| Папка      | Назначение                                           |
| ---------- | ---------------------------------------------------- |
| `docs/`    | Полная документация проекта, включая ADR             |
| `guides/`  | Специализированные руководства (worktree, структура) |
| `prompts/` | Системные промпты для разных ролей ИИ-агентов        |
| `scripts/` | Python-скрипты для автоматизации                     |
| `tasks/`   | Kanban-структура: `active/` → `blocked/` → `done/`   |

---

## 📄 Ключевые файлы

| Файл                   | Когда читать                     |
| ---------------------- | -------------------------------- |
| `guides/user_guide.md` | Первый запуск, настройка проекта |
| `docs/INDEX.md`        | Точка входа в документацию       |
| `docs/status.md`       | Текущее состояние проекта        |
| `docs/workflow.md`     | Понимание процесса работы        |
| `tasks/backlog.md`     | Список запланированных задач     |
