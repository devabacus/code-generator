# Финализация дискуссии

Отвечай на русском.

Ты завершаешь дискуссию с User. Твоя задача — зафиксировать результаты в документации проекта.

> **Важно:** Все пути указаны относительно папки `ai/`.
> Если ты находишься в `discussions/`, используй `../ai/docs/...`

## Что сделать

1. **Прочитай текущие документы (в папке `ai/`):**

   - `ai/docs/architecture.md`
   - `ai/docs/roadmap.md`
   - `ai/docs/status.md`
   - `ai/docs/agent_memory.md`

2. **Обнови документы по результатам дискуссии:**

   | Документ          | Что обновить                                    |
   | ----------------- | ----------------------------------------------- |
   | `architecture.md` | Новые решения по архитектуре, стеку, структуре  |
   | `roadmap.md`      | Изменения в фазах, приоритетах, фичах           |
   | `status.md`       | Текущая фаза, риски, следующий фокус            |
   | `agent_memory.md` | Gotchas, предпочтения User, технические заметки |

3. **Создай ADR если было принято важное решение:**

   - `ai/docs/decisions/adr-XXXX-название.md`

4. **Резюмируй что обновил** — кратко для User.

## Правила

- Не перезаписывай — **дополняй** существующую информацию
- Не добавляй спекуляции — только то, что **явно обсуждалось**
- Если не уверен — **спроси User**

## Пример использования

```
Прочитай prompts/finalize.prompt.md

Мы обсудили:
- Переход на PostgreSQL вместо SQLite
- Добавление фазы "Миграция данных" в roadmap
- Риск: нужна проверка совместимости ORM

Обнови соответствующие документы.
```
