## Команды

Примеры для разных стеков — адаптируй под свой проект.

### Node.js (pnpm/npm/yarn)

```bash
pnpm dev          # Запустить dev-сервер
pnpm test         # Запустить тесты
pnpm lint         # Проверить код
pnpm build        # Production-сборка
```

### Python

```bash
python -m pytest           # Тесты
python -m pytest --cov     # С покрытием
ruff check .               # Линтинг
python main.py             # Запуск
```

### Flutter/Dart

```bash
flutter run                # Запуск
flutter test               # Тесты
dart analyze               # Анализ кода
flutter build apk          # Сборка
```

### Go

```bash
go run .                   # Запуск
go test ./...              # Тесты
golangci-lint run          # Линтинг
go build                   # Сборка
```
