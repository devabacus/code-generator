# code-generator instructions
Это расширение генерирует новые проекты и т.д. Основной принцип оно берет проект-шаблон и через замену (чаще всего это замена названия проекта) создает новый копированием, походу может выполнять разные терминальные команды.

Это расширение только в крайнем случае генерирует файлы на разных языках для разных проектов, в основном всегда приоритет это копирование файлов из шаблона проекта (файлы dart, py, ts, etc). Если ты хочешь захардкодить создание файла на другом языке, то сначала спрашиваешь у меня.
